# КАСКАДВЕБ

Веб-страница с визуальным представлением изображения "Лес вдали", оформленного в едином стиле с баннером и интерактивными элементами.

## Архитектура

Проект представляет собой простую одностраничную статическую веб-страницу (SPA). Структура основана на HTML, стилизация выполнена с помощью CSS. Страница разделена на две основные части: верхний баннер и карточка изображения. Стили определены в отдельном файле `styles.css`.

## Технологический стек

*   **HTML5**: Семантическая разметка.
*   **CSS3**: Стилизация, Flexbox для центрирования и расположения элементов, переменные CSS, псевдоэлементы, медиазапросы для адаптивности.
